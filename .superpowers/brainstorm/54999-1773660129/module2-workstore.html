Work Status State Machine

draft → creating → ready → publishing → published
draft: 刚创建  |  creating: Agent 正在生成  |  ready: 所有步骤完成,待发布  |  publishing: Playwright 发布中  |  published: 已上线
Any state can → failed (with error message)
