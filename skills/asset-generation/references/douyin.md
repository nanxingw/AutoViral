# 抖音素材生成参考

## 分辨率与宽高比规格

| 用途 | 宽高比 | 像素 (w×h) | 备注 |
|------|--------|-----------|------|
| 标准视频 | 9:16 | 1080×1920 或 1088×1920 | 抖音最常见的格式 |
| 电影感视频 | 16:9 | 1920×1080 | 用于电影感/横屏内容 |
| 头像 | 1:1 | 1080×1080 | 各平台通用 |

**API 安全值（64 的倍数）：**
- 9:16 → 1088×1920
- 16:9 → 1920×1088

## 视频生成细节

抖音是一个**短视频平台**——所有内容都围绕视频片段展开（通常 15 秒到 60 秒，最长可达数分钟）。

### 首帧 + 视频工作流程

抖音素材生成的标准流程：
1. 为每个镜头生成一张 9:16 (1088×1920) 的**首帧图片**
2. 用首帧生成**视频片段**（每个镜头约 5 秒）
3. 视频片段后续在 content-assembly 步骤中进行合成

### 抖音视频分辨率说明

- **9:16 竖屏**是默认且主流的格式——除非分镜明确要求电影感/横屏，否则一律使用竖屏
- **16:9 横屏**用于电影感内容、知识分享或纪录片风格的视频
- 生成视频首帧图片时，宽高比必须与目标视频的宽高比一致

## 抖音专属风格关键词

针对抖音平台优化的短视频内容：

**抖音通用审美：**
```
cinematic, dynamic composition, vibrant colors, social media optimized, short-form video aesthetic, high production value
```

**抖音流行风格：**
```
trendy, eye-catching, scroll-stopping, bold visual, Chinese social media aesthetic
```

**Vlog/日常风格：**
```
vlog style, casual, authentic feel, lifestyle content, relatable, first-person perspective
```

**知识类风格：**
```
clean layout, clear visual hierarchy, informational, professional, trust-building aesthetic
```

## 抖音专属提示词技巧

1. **首帧至关重要**——抖音用户在前 0.5 秒内决定是否继续观看。首帧必须视觉冲击力强，并能立刻传达视频的价值。

2. **竖屏优先构图**——始终按 9:16 构图。主体放在画面中心或上三分之一处。重要元素避开顶部 10% 和底部 20%（这些区域会被 UI 覆盖）。

3. **大胆且高对比度**——抖音内容在快速滑动的信息流中竞争。使用强对比度、鲜艳色彩和清晰的主体。除非内容类型需要，否则避免过于含蓄或暗淡的视觉风格。

4. **为运动预留空间的首帧**——生成视频首帧时，要考虑预期的镜头运动。为摄像机移动留出"空间"（例如，如果计划做推远效果，首帧就用更紧凑的构图）。

5. **文字安全区域**——抖音会在底部叠加字幕，右侧有互动按钮。主体构图时要注意避开这些区域。
