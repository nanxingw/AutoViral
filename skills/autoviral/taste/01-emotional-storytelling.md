# 情感与叙事

## 核心命题

短视频不是"几个镜头拼起来"，是**一段情感旅程**。观众在几秒到几十秒内要经历一段微型心理曲线——从"咦？"到"啊！"到"发"。没有这条曲线，再精致的画面也是噪声。

好作品的共同点，都是**先有情感设计，再有视觉执行**。反过来做的，都沦为素材展示。

## 情感意图（Emotional Intent）

开工第一个问题：**你想让观众感到什么？**

不是"感到我的品牌"、"感到这个产品好"——那是效果，不是情感。情感是观众身体的反应。

### 情感词表

用这些具体的词，不要用"好看"、"有趣"、"调性好"：

| 类别 | 词汇 | 常见触发 |
|---|---|---|
| **向上** | 好奇、惊喜、敬畏、温暖、治愈、心动、释然、自豪 | 反常识、美、共鸣、transformation、超越 |
| **向下** | 紧迫、焦虑、不甘、嫉妒、心疼、愤怒、遗憾、恐惧 | 冲突、压力、对比、失去、危险 |
| **混合** | 荒诞、讽刺、黑色幽默、苦涩的甜、哭笑不得 | 反差、矛盾、倒错期待 |

### 单一主导原则

**一个作品只能有一个主导情感**。可以有铺垫情感（A → B 的转化），但**落点只能有一个**。同时想让人又感动又狂笑，通常两头落空。

如果 brief 里写了"希望观众感到感动、治愈、温暖、有共鸣、也想马上行动"——这不是五个情感，这是"我不知道我想要什么"。逼自己缩到一个词。

### 反模式

- ❌ "让观众感受到品牌调性" → 这不是情感，是 briefing 术语
- ❌ "有趣、好玩" → 太泛，十个人做出来会是十个方向
- ❌ "想让他们点赞" → 点赞是结果，情感是原因
- ✅ "让观众感受到：当他看到一个从未想到过的视角时，那种一瞬间脑子发麻的好奇"

## 叙事弧（The Arc）

五段式基础结构，不管 7 秒还是 70 秒都适用：

```
钩子 (Hook) → 语境 (Context) → 证明/升级 (Proof/Escalation) → 回报 (Payoff) → 落点 (Landing)
```

### 1. 钩子（Hook，0-3 秒）

**目的**：回答观众心里"我为什么不划走？"

前 2-3 秒内必须完成。可用手段：

- **视觉冲击** — 反常规画面、饱和色、极端构图、快速运动
- **认知冲击** — 违反常识的陈述、反直觉数据、悬念问题
- **冲突陈述** — "他们都说 X，但其实 Y"
- **情境钩** — 把观众放进一个他立刻能代入的场景
- **身份钩** — "如果你也 XX，这条必看"（慎用，容易 cheap）

**红线**：钩子不是后面内容的概述。**钩子是抛承诺，不是剧透**。"今天教你三个做饭技巧" 是剧透，"我 30 秒切好一颗洋葱还不流泪" 是钩子。

### 2. 语境（Context，3-8 秒）

**目的**：让观众知道"这跟我有什么关系"。

迅速建立：**谁**、**在哪**、**为什么这件事重要**。

这里是大多数新手拖戏的地方——短视频里**语境必须极简**，能用一个镜头带出的就别用三个。Stillmotion 的 **People-Place-Plot-Purpose** 框架很好用：一句话交代四件事。

### 3. 证明 / 升级（Proof / Escalation）

**目的**：兑现钩子承诺，同时把情感强度推向峰值。

两种基本模式：

- **证明型** — 钩子是断言（"这个方法能 5 分钟解决 X"），主体逐步验证
- **升级型** — 钩子是微冲突，主体中冲突持续加码（压力越来越大）

**节奏原则**：这一段**必须有节奏变化**。全程匀速 = 情感平线 = 观众划走。参考 `taste/03-rhythm-and-editing.md`。

### 4. 回报（Payoff）

**目的**：给观众一个情感释放的瞬间。

Payoff 是作品的真正价值所在——没有 payoff 的视频像没有包袱的相声。形式不限：

- 视觉惊艳的成果展示
- 反转的真相揭示
- 方法被验证后的"原来如此"
- 情感积累后的宣泄

**Payoff 必须被剪辑标记出来**。用停顿、用音乐 drop、用画面定格、用近景特写——观众必须"知道"这是高潮。藏着掖着等于没 payoff。

### 5. 落点（Landing）

**目的**：让观众带着情感离开，而不是被硬塞一个"点赞关注"。

三种好的落点：

- **余韵型** — 最后一个画面停留稍长，让情感沉淀
- **余味型** — 一句话总结或一个小幽默，让人回甘
- **下一件事型** — 自然引出下一个可以看的东西

**反模式**：机械的"记得点赞关注评论分享"是最差的落点。观众会嫌弃。要让行动呼吁融进情感：
"如果你也想做出这样的画面，评论区我教你。" 比"点赞关注"好十倍。

## 叙事结构的变体

不是每个作品都走完整五段。常见变体：

| 结构 | 适用 | 示意 |
|---|---|---|
| **纯钩 + 回报** | 超短素材（<10s） | 钩子 → 直接 payoff |
| **反转型** | 悬念 / 黑色幽默 | 钩 → 铺垫期待 → 打破期待 |
| **并置型** | 教学 / 对比 | 钩 → A vs B → 结论 |
| **节拍型** | 舞蹈 / 混剪 | 钩 → 音乐驱动的多段回报 |
| **纪实型** | vlog / 记录 | 钩 → 过程 → 某个小瞬间 |
| **情绪滑坡型** | 情感类 | 钩 → 步步积累 → 情绪决堤 |

## 自检清单

产出 brief 或脚本后，逐条回答：

- [ ] 我能用**一个**具体情感词说出这个作品想让观众感到什么
- [ ] 前 3 秒有明确的钩子，不是摘要、不是慢热、不是自我介绍
- [ ] 每个镜头 / 段落都在推动情感或叙事，没有"填充"、"转场需要"之类的废镜头
- [ ] 有一个可辨识的 payoff 瞬间，观众能感受到这是高潮
- [ ] 落点有情感余韵，不是生硬的 CTA
- [ ] 如果我自己在刷关注流，我会停下来看完这个

任何一项答不上来——回到第一条重想，不要前进。
