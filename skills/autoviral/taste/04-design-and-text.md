# 设计与文字

## 核心命题

短视频里的**每一帧静态画面都是一张海报**。每一个文字叠加都是一次排版决策。封面图、字幕、信息图、图文笔记——所有静态视觉都遵循同一套设计语法。

糟糕的设计**不会被观众明确吐槽**，但会让作品看起来"差点意思"——而差一点就是差很多。

## 视觉层级（Visual Hierarchy）

观众的眼睛会扫描，不会阅读。**在 1 秒内你必须告诉他"先看哪里"**。

### 建立层级的四种手段

按影响力排序：

1. **大小（Size）** — 最重要的元素最大。差异要大——主标题字号是副文字号的 2-3 倍，不是 1.2 倍。
2. **对比（Contrast）** — 颜色、明度、饱和度的反差。重点元素与背景对比越强越跳出。
3. **位置（Position）** — 视线会按 F 型 / Z 型扫描。左上、中心是焦点位。
4. **留白（Negative Space）** — 一个元素周围空得越多，它越显得重要。

### 三点原则

一张画面里**最多三个视觉焦点**。超过 = 视觉混乱。

- 一个**主焦点**（视线第一落点）
- 一个**次焦点**（补充信息）
- 一个**尾焦点**（Logo / 来源 / CTA）

超过三个 = 观众什么都记不住。

## 字体（Typography）

### 选字原则

- **一个作品最多两种字体**：一个标题、一个正文。超过 = 廉价感。
- **标题字要有个性**：衬线、非衬线、手写、展示体，选一个匹配情感。
- **正文字要无感**：越"透明"越好，让读者不会意识到字体的存在。
- **中英混排**：中文选择一套对应西文字重接近的英文字，不要让英文看起来飞起来。

### 中文字体情感对照

| 情感 | 字体类型 | 例 |
|---|---|---|
| **权威 / 严肃** | 衬线（宋体类）、粗笔画 | 思源宋体、方正风雅宋 |
| **现代 / 专业** | 几何无衬线 | 思源黑体、苹方、OPPO Sans |
| **温暖 / 柔和** | 圆体、手写感 | 阿里巴巴普惠体、手写小楷 |
| **冲击 / 强烈** | 极粗展示体 | 方正兰亭粗黑、各种厂牌展示字 |
| **复古 / 怀旧** | 老式宋体、书法、字库特色字 | 汉仪旗黑、演示体 |

### 字号规则

- **主标题**：在目标屏幕尺寸下，字高不小于画面高度的 8%
- **正文**：不小于画面高度的 3%
- **腿部信息（Logo / 出处）**：不小于 1.5%（但别更小——小到看不清等于没有）

### 易读性红线

- **字和背景对比度必须足够**。白字直接压在浅色视频上 = 灾难。加阴影、加描边、加半透明底色——三选一，别叠。
- **字不要太长**。单行超过 18 个汉字 / 35 个英文字符 = 观众读不完。
- **关键字不要放在屏幕最底部 10%**。平台 UI 会遮挡。

## 色彩（Color）

### 色彩情感

| 色系 | 情感 | 适用 |
|---|---|---|
| **暖色（红、橙、黄）** | 能量、紧迫、热情、温暖 | 行动号召、美食、节日 |
| **冷色（蓝、青、紫）** | 冷静、专业、科技、忧郁 | 工具、教程、夜景 |
| **中性（黑白灰棕）** | 克制、高级、永恒 | 时尚、纪实、艺术 |
| **高饱和荧光** | 冲击、年轻、不羁 | 街头、Z 世代、潮玩 |
| **低饱和莫兰迪** | 温柔、质感、治愈 | 生活方式、家居、美妆 |

### 配色原则

- **主色 1 + 辅色 1 + 点缀色 1**。三种颜色够了。
- **点缀色与主色形成对比**——互补色或高亮度差。这是视觉"呼吸口"。
- **整部作品色调统一**。一段冷调、一段暖调来回切换 = 分裂感（除非你在刻意做对比叙事）。

### 反模式

- ❌ 彩虹七色堆叠 → 嬉皮士滤镜
- ❌ 大量撞色 → 眼睛累
- ❌ 低饱和 + 低对比 + 灰蒙蒙 → 看起来像没调色

## 文字动画（Kinetic Text）

当文字出现在动态视频里：

- **一次一句话**。一屏不要同时显示两段话。
- **出现节奏与口播对齐**。听到关键词的瞬间，关键词出现。
- **不要整段淡入淡出**。逐字出现、打字机效果或位移入场都更有节奏感。
- **强调的关键词放大或变色**。但别每个词都强调——都强调 = 都不强调。

## 封面 / 单帧海报设计

用于抖音封面、小红书首图、YouTube 缩略图等：

### 必做清单

- [ ] 主焦点在画面中心偏上 1/3 位置
- [ ] 主标题占画面高度 10-15%（移动端可读的最小尺寸）
- [ ] 配色不超过 3 种主色
- [ ] 留出底部 20% 作为平台 UI 安全区
- [ ] 如果有人物，眼神看向主文字或画面内（不要看向画外）
- [ ] 缩到缩略图尺寸（150px 宽）仍能看清主标题

### 红线

- ❌ 把所有信息都塞上去 → 观众一眼扫过什么都记不住
- ❌ 用平台默认字体 + 居中 → 毫无个性的"AI 模板感"
- ❌ 过度 kitsch（各种贴纸、边框、emoji） → 2015 年 PhotoShop 味

## 一帧设计自检

对任一静态画面问：

- [ ] 1 秒内我能说出"这张图在讲什么"吗？
- [ ] 视线是从哪里开始、按什么顺序走的？这条路径是我设计的吗？
- [ ] 主标题和背景的对比足够让最差的屏幕也看得清吗？
- [ ] 整个作品里，这一帧和其他帧是同一个视觉世界吗？
