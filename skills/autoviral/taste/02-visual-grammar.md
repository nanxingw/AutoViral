# 视觉语法

## 核心命题

每一个镜头的**大小、机位、运动、构图**都是一句话——它在对观众说话。你不是在"填一个画面"，你是在用视觉选词。

选错了，情感就错了。再精致的渲染也救不回来。

## 镜头大小（Shot Size）

决定观众与对象之间的**心理距离**。

| 尺寸 | 信息密度 | 情感 |
|---|---|---|
| **极远景（Extreme Wide）** | 环境为主，主体极小 | 敬畏、孤独、史诗感 |
| **远景（Wide）** | 环境 + 主体 | 语境建立、小开大合 |
| **中景（Medium）** | 主体腰部以上 | 中性、对话、叙事 |
| **近景（Close-up）** | 面部、细节 | 亲密、共情、紧张 |
| **极近景（Extreme Close-up）** | 眼睛、嘴唇、物件细节 | 极度强调、仪式感 |

### 实操原则

- **开场**优先用远景或极近景——两种极端都能抓人。中景开场最容易被划走。
- **情感峰值**几乎总是近景或极近景。没有例外。
- **一个段落内**镜头大小要有起伏（大 → 小 → 大），匀速观看疲劳。

## 机位（Camera Angle）

### 五种基础机位

- **平视（Eye-level）** — 中性、平等、纪实。默认不怕的情况下用这个。
- **低机位（Low Angle）** — 主体显得强大、有压迫感、神圣。
- **高机位（High Angle）** — 主体显得弱小、脆弱、无助。
- **俯视（Top-down）** — 上帝视角，客观、秩序、禅意（平铺美食、工艺步骤）。
- **荷兰角（Dutch Angle）** — 世界歪了。用在不安、错乱、惊悚，谨慎。

### 红线

- **机位不是装饰**。给普通对话加低机位，观众会觉得你在炫技。
- **荷兰角每个作品最多用一次**。用多了观众会晕。
- **高机位用在主角身上要小心**——会让观众潜意识觉得"我在俯视这个人"，如果你想让观众喜欢他就别用。

## 运动（Camera Movement）

运动是情感的加速器。**什么时候让相机动 = 什么时候让情感提速**。

| 运动 | 情感作用 | 典型用法 |
|---|---|---|
| **Static（静止）** | 沉静、专注、审视 | 让观众看清 / 感受节奏 |
| **Push-in（推进）** | 关注度↑、情感↑ | 揭示、强调、"接下来重要" |
| **Pull-out（拉远）** | 格局↑、孤独↑ | 收尾、交代全貌、对比 |
| **Pan / Tilt（摇）** | 揭示、引导目光 | 跟随视线、渐进呈现 |
| **Tracking（跟随）** | 沉浸、动能 | 主角行进、过程感 |
| **Handheld（手持）** | 在场、真实、慌乱 | 纪实、冲突、紧迫 |
| **Drone / Crane（航拍/升降）** | 格局、史诗、上帝 | 开场、转场、收尾 |
| **Dolly Zoom** | 失真、精神冲击 | 极端情绪瞬间，年度最多一次 |

### 运动的节奏原则

- **一个镜头内不要超过一个主要运动**。推 + 摇 + 跟 = 油腻。
- **运动的起始和结束都要有 beat**。突然起、突然停 = 廉价感。
- **AI 生成视频的限制**：Dreamina 和同类工具擅长 push-in 和缓慢静止，不擅长剧烈手持和复杂 tracking。扬长避短，给它设计能发挥的运动。

## 构图（Composition）

### 核心工具箱

- **三分法（Rule of Thirds）** — 主体不居中，放在三分线交点。默认用这个。
- **引导线（Leading Lines）** — 路、墙、栏杆、目光方向，把观众视线牵到主体上。
- **对称（Symmetry）** — 稳重、仪式感。用在需要秩序/权威/神圣的场景。
- **负空间（Negative Space）** — 主体周围留白。越少画面 = 越强情感（孤独、专注）。
- **框中框（Frame within Frame）** — 用窗、门、屏幕等框出主体。增加层次与窥视感。

### 反模式

- ❌ 主体完美居中 + 背景密集 + 无层次 = 证件照感
- ❌ 想装文艺就负空间 → 没有情感意图的负空间等于空洞
- ❌ 多重引导线指向不同位置 → 视线打架
- ❌ 对称构图 + 不对称内容（一边多一边少） → 观众潜意识难受

## 情感 → 技术映射表

给定情感意图，优先选择：

| 情感意图 | 镜头大小 | 机位 | 运动 | 构图 |
|---|---|---|---|---|
| **敬畏/惊叹** | 极远景、极近景 | 低机位或俯视 | 缓慢 push-in / pull-out / crane | 对称或极端负空间 |
| **亲密/温暖** | 近景 | 平视或微低 | 微 push-in、静止 | 浅景深，三分法 |
| **紧迫/焦虑** | 中近、紧构图 | 平视或荷兰 | 手持、抖动 | 框挤压，负空间极少 |
| **反转/惊讶** | 多尺寸切换 | 切换机位 | 快速推 / rack focus | 打破前一段节奏的构图 |
| **动能/活力** | 中近 | 平视 | 跟随、whip pan | 强对角线 |
| **孤独/空旷** | 极远景 | 俯视或平视 | 静止或缓慢 pull-out | 大量负空间 |
| **沉静/克制** | 中景 | 平视 | 静止 | 对称或三分 |
| **诡谲/不安** | 近景局部 | 荷兰 | 缓慢推或 dolly zoom | 框中框、不对称 |

## AI 生成的视觉语法特殊考虑

当画面由 AI 生图/生视频产出时，这些原则**仍然适用**，只是实现路径不同：

- 用 prompt 里的 "wide shot of…" / "extreme close-up of…" 显式指定镜头大小
- 用 "low angle" / "eye level" / "top-down view" 指定机位
- 用 "slow push-in" / "static" / "handheld documentary" 指定运动
- **一张图里只给一个主要构图指令**，别让 prompt 打架（"rule of thirds + centered symmetry" = 模型懵圈）
- AI 默认产出趋向"证件照中景 + 居中构图"——你必须明确对抗这个默认

## 一帧自检

对任一生成的画面问：

- [ ] 这个镜头大小服务于什么情感？换一个尺寸会丢失什么？
- [ ] 机位是中性的还是带态度的？带态度的，那是正确的态度吗？
- [ ] 运动有明确目的，还是只是"动一下免得太闷"？
- [ ] 构图里视线最后落在主体上吗？
- [ ] 如果把这一帧截图单发，它自己能传达情感吗？

答不上来的镜头 = 废镜头，重生或剪掉。
