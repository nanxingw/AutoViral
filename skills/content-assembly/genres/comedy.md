# 搞笑/抽象类 — 合成阶段专项指南

本文档覆盖搞笑（Comedy）和抽象（Abstract）品类在内容合成阶段的剪辑节奏、BGM 策略、音效使用和转场规则。

---

## 搞笑类剪辑节奏

- **铺垫阶段**：正常节奏，每个镜头 2-4 秒
- **反转瞬间**：突然加速或突然慢放
- **点睛阶段**：定格或慢动作 + 配乐变化
- **跳切（Jump Cut）**：标配剪辑手法——去掉犹豫和废话，只留最精华的反应

---

## 搞笑类 BGM 与音效

### BGM 四种战术用法

**1. 情绪铺垫-反转**
```
[铺垫] 史诗感BGM / 煽情钢琴曲 / 紧张悬疑音乐
[反转瞬间] 突然静音 0.5秒 → 切换搞笑音效/沙雕BGM
```

**2. 卡点强化**
```
[动作1] 卡在第1拍
[动作2] 卡在第2拍
[反转动作] 卡在重音拍 → 停顿
```

**3. 反差配乐**

| 画面 | BGM | 效果 |
|------|-----|------|
| 菜市场砍价 | 商战电影配乐 | 荒诞感 |
| 起床倒水 | 史诗战争音乐 | 把小事拍出大片感 |
| 被老板骂 | 轻快的儿歌 | 黑色幽默 |

**4. 经典梗音乐**

抖音有大量"一听就知道要搞笑"的条件反射式 BGM。要跟踪当下流行的梗音乐，过气的音乐反而减分。

### 音效精确使用

| 音效类型 | 使用时机 | 效果 |
|---------|---------|------|
| 急刹车音效 | 反转瞬间 | "等一下，什么？" |
| 新闻联播前奏 | 严肃语气播报荒诞内容 | 正式感反差 |
| "噔"/"叮" | 角色意识到什么的瞬间 | 内心活动外化 |
| 拳击/打击音效 | 夸张肢体动作配合 | 强化物理喜剧 |
| 静音 0.3-0.5秒 | 反转前 | "暴风雨前的宁静" |
| 慢放音效 | 社死/尴尬瞬间 | 延长尴尬变成笑点 |

### BGM 原则

1. **节奏服从内容**：先定剪辑节奏，再选匹配的音乐
2. **热门 BGM 有流量加成**：抖音算法对"同款音乐"内容有推荐倾斜
3. **音量动态控制**：台词部分 BGM 降到 15-25%，纯画面部分可到 40-60%
4. **反转点必须有声音标记**：静音、音效、换曲——至少用一种

## 抽象类剪辑策略

抽象内容的剪辑和搞笑类有显著区别：

- **长镜头往往更有力**：让荒诞场景持续存在，不剪掉，观众的困惑会累积
- **不需要"反转剪辑点"**：抽象内容从头到尾都是错位的
- **重复/循环**：同一个荒诞动作循环出现，强化"他是认真的"感觉

---

## 抽象类 BGM 规则

BGM 在抽象内容中不是配角，而是构成抽象的核心元素之一。

### 三种 BGM 策略

**1. BGM 作为错位的一端**

| 画面 | BGM | 错位类型 |
|------|-----|---------|
| 赛博朋克/暗黑 | 儿歌/可爱 | 硬 vs 软 |
| 欧美街头 | 凤凰传奇/广场舞 | 洋 vs 土 |
| 高奢时装秀 | 唢呐/二人转 | 雅 vs 俗 |
| 田园牧歌 | 死亡金属 | 静 vs 燥 |

**关键**：BGM 也要"纯正"。

**2. BGM 强化"过度认真"**

用和事件规模完全不匹配的史诗/庄严音乐。

**3. BGM 强化"过度随意"**

用极度轻快/无所谓的音乐消解事件的严重性。

### 音效策略

抽象内容对音效的依赖低于搞笑类。特有用法：

**沉默**：在最荒诞的瞬间，不加任何音效。因为抽象内容的态度是"这很正常"——加搞笑音效等于在告诉观众"这里该笑了"，破坏了"一本正经"的氛围。

## 搞笑/抽象类发布文案规则

**硬性规定：文案必须是简短的一句话。** 满足以下任一形式：

1. **讽刺/搞笑评论**：用一句话吐槽或评论视频里的内容
2. **引导观看**：用一句话勾起好奇心，引导用户去看视频
3. **自我代入**：以视频中主体人物的第一人称说一句话

**绝对禁止：**
- 不许剧透视频里的笑点
- 不许解释视频在演什么
- 不许写超过一句话的长文案
- 不许用"哈哈哈"等语气词凑字数

**示例：**
- 讽刺型：`"确认过眼神，是甲方看了会沉默的方案"`
- 引导型：`"看到最后我直接笑出了猪叫"`
- 代入型：`"我当时真的觉得自己很帅"`
