# 搞笑/抽象类 — 策划阶段专项指南

本文档覆盖搞笑（Comedy）和抽象（Abstract）品类在内容策划阶段的结构公式、Hook 设计、台词规则和自检清单。

---

## 搞笑类的七种底层结构

### 1. 经典反转型（最通用，成功率最高）

```
铺垫A → 铺垫B（强化预期） → 反转C（打破预期） → [可选] 点睛D
```

- **核心**：前面越"正经"，反转越好笑
- **关键**：反转必须在逻辑上说得通，但在情感上出乎意料
- **时长**：15-30秒最佳

### 2. 递进荒诞型（适合系列化）

```
正常 → 稍微离谱 → 更离谱 → 彻底离谱
```

- **核心**：每一层都比上一层荒诞，但每一步都有内在逻辑
- **关键**：递进节奏要快，不能在任何一层停留太久

### 3. 错位/身份反差型

```
角色A进入不属于他的场景B → 用A的逻辑处理B的问题
```

### 4. 重复-打破型

```
模式A重复3次 → 第4次打破模式
```

- **核心**：喜剧的"三次法则"（Rule of Three）

### 5. 平行对比型

```
[情境A的反应] vs [情境B的反应]（同一个人，反差巨大）
```

### 6. 紧张-崩塌型

```
严肃/紧张/恐怖的氛围持续升温 → 突然以最不严肃的方式崩掉
```

### 7. 微观共鸣型（观察式喜剧）

```
一个极其微小但普遍存在的私密体验 → 精确还原
```

- 不需要复杂结构，一个场景就够
- 时长：10-30秒

## 抽象类的七种结构模式

### 1. 感官错配型
元素A的视觉 + 元素B的听觉。两端都要"纯正"。

### 2. 过度认真型
微不足道的小事 + 史诗级的处理方式。演员必须一本正经。

### 3. 过度随意型
重大的事 + 极度轻描淡写的态度。

### 4. 语境位移型
角色A + 不属于A的场景B → 用A的规则生活在B里。角色不能"意识到"自己不属于这里。

### 5. 形式解构型
观众熟悉的视频套路 + 完全歪掉的执行。必须先让观众以为是"正经版"。

### 6. 真实解构型
日常场景 + 字幕/旁白揭示"真实含义"。

### 7. 逻辑自洽的平行宇宙型
建立一套错误但完整的世界观 → 在这套规则里严格运行。

## Hook 设计（前3秒）

### 搞笑类 Hook

| Hook 类型 | 原理 | 示例 |
|-----------|------|------|
| **视觉荒诞** | 画面本身就不对劲 | 西装男坐在马桶上开会 |
| **反常识开场白** | 台词打破认知 | "今天教大家如何在30秒内被开除" |
| **结果前置** | 先看到离谱的结果 | 先看到满脸奶油的人，再倒叙 |
| **紧张感制造** | 假装严肃/危机 | 用新闻联播的语气播报"舍友又偷吃我零食" |
| **场景标题（共鸣型专用）** | 文字描述场景 | "当你躺在床上玩手机的时候" |

**禁忌：** 不要用"大家好今天给大家分享"；不要前3秒还在铺环境。

### 抽象类 Hook

| Hook 方式 | 原理 |
|-----------|------|
| **画面即错位** | 第一帧就不对劲 |
| **声音即错位** | 第一个音符就不属于这个画面 |
| **行为即错位** | 角色第一个动作就不合理 |
| **文字即错位** | 字幕和画面形成反差 |

**区别**：搞笑类 Hook 制造好奇心，抽象类 Hook 直接制造困惑。

## 台词规则

### 搞笑类

1. **短句为王**：每句不超过10个字
2. **口语化**："我靠这也行？" 比 "这真是出乎意料" 好笑100倍
3. **留白**：最好笑的瞬间往往不需要台词
4. **文案节奏公式**：短句 + 短句 + 短句 + ——长句（反转/点睛）

**字幕铁律：没有人说话的时候，不加任何字幕。**

| 字幕用法 | 效果 |
|---------|------|
| 吐槽式旁白 | "上帝视角"笑点 |
| 反差字幕 | 画面和文字说的是两回事 |
| 翻译字幕 | 把客套话"翻译"成真实含义 |
| 倒计时/标注 | 预告即将发生的事 |

### 抽象类

| 台词模式 | 说明 |
|---------|------|
| 正经话说荒诞事 | 用完全正式/专业的语言描述荒诞的内容 |
| 无台词——纯氛围 | 错位通过画面、动作和音乐传达 |
| 角色内部自洽 | 角色说的话在他自己的世界里完全合理 |

**抽象类无人说话时绝对不加字幕**——反差本身就是全部信息，字幕=解释笑点=破功。

## 自检清单

### 通用检查
- [ ] 前3秒能抓住注意力？
- [ ] 能让目标观众笑出来？
- [ ] 有"想转给朋友"的冲动？
- [ ] 没有引起不适的内容？
- [ ] 总时长合理？（搞笑 15-45秒，抽象 10-30秒）

### 搞笑类额外
- [ ] 有明确的预期建立阶段？
- [ ] 有至少一个有力的反转？
- [ ] 铺垫不超过总时长的 60%？
- [ ] 反转点有声音标记？

### 抽象类额外
- [ ] 错位足够极端？
- [ ] 每个属性的纯度够高？
- [ ] 创作者态度始终认真？
- [ ] 无人说话的片段没有加字幕？

## 避坑指南

### 搞笑类
| 错误 | 正确做法 |
|------|---------|
| 铺垫太长 | 铺垫控制在总时长 40-60% |
| 反转太弱 | 反转要超出预期 |
| 笑点需要解释 | 如果要加字幕解释才好笑，重写 |
| 表演过度 | "一本正经地做荒诞的事"才是最高级的搞笑 |

### 抽象类
| 错误 | 正确做法 |
|------|---------|
| 错位不够极端 | 两端都要拉到极致 |
| 纯度不够 | 每个元素单独看都要是正宗的 |
| 自己先笑了 | 全程保持认真 |
| 加了搞笑音效 | 用沉默或与错位方向一致的音效 |

## 发布文案规划

策划阶段就要确定文案方向。搞笑/抽象类文案**必须是简短的一句话**，三种形式选其一：

1. **讽刺/搞笑评论** — 吐槽视频内容
2. **引导观看** — 勾起好奇心
3. **自我代入** — 以视频主体第一人称说话

**禁止剧透笑点、禁止解释剧情、禁止长文案。**
