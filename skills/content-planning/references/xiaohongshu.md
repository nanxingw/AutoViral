# 小红书（XHS）内容策划参考

## 小红书内容策略

**什么类型的内容表现好：**
- 颜值优先——精美的封面图决定点击率
- 有教育价值——"干货"内容容易被收藏
- 真实的个人表达——亲身经验 > 泛泛建议
- 清单格式——"10个..." "5种..." 结构化内容
- 配图清晰的分步教程
- 优缺点俱全的详细评测

## 图片规格

- 封面图：推荐 1080×1440（3:4），或 1080×1920（9:16）
- 内页图片：整篇帖子保持统一比例
- 最少 3 张图，推荐每篇帖子 6-9 张图
- 图片质量：清晰、光线好、美观

## 小红书图片内容原则

- 第一张图就是封面——它决定了点击率
- 封面应具备：清晰的主体 + 可读的标题文字 + 好看的背景
- 所有图片保持统一色调
- 图片上的文字在手机端必须可读（最小相当于 24px）
- 加入一张"拍摄信息"或"详细信息"图以增加可信度

## 小红书节奏模板

**种草型（Product showcase/小红书，15-30s）：**
```
[Hook: 痛点 3s] → [产品介绍 5s] → [使用展示 5-8s] → [效果对比 5s] → [总结推荐 3s] → [CTA 3s]
```

## 小红书输出格式：短视频发布信息

```markdown
### 小红书
- **标题:** [SEO 优化标题，含关键词和 emoji]
- **正文:** [帖子正文]
- **标签:** #tag1 #tag2 ... #tag10
- **话题:** #topic1 #topic2
```

## 小红书输出格式：图文发布信息

```markdown
### 小红书
- **标题:** [含关键词、emoji 的标题，建议不超过 20 字]
- **正文:**
  [完整帖子正文——结构化，有换行、emoji，自然融入相关关键词。最佳表现长度为 300-800 字。]
- **标签:** #tag1 #tag2 ... #tag10
- **话题:** #topic1 #topic2 #topic3
```

## 小红书平台注意事项

- 小红书是 SEO 驱动型平台——标题和正文应自然包含相关关键词
- 封面图需要留出干净的区域用于叠加标题文字
- 莫兰迪色在小红书上特别受欢迎，适合高端/美学类内容
- 小红书用户看重真实感——避免过度精修的、"广告感"太强的内容
- 在小红书上，图文帖的整体表现通常优于视频
- 标签和话题对内容被发现至关重要——最多使用 10 个标签和 3 个话题
