# 抖音（Douyin）内容策划参考

## 抖音内容策略

**什么类型的内容表现好：**
- 钩子驱动型内容——首帧必须让人停住滑动
- 热门 BGM——使用热门音频能获得算法加权
- 话题挑战——参与热门挑战活动
- 合拍/对口型——与已有爆款内容互动
- 带有反转结局的情感故事
- 节奏明快、步骤清晰的实用教程

## 视频规格

- 分辨率：1080×1920（9:16）或 1088×1920
- 时长：7-60 秒为最佳（算法偏好完播率高的内容）
- 帧率：30fps 标准，60fps 适用于高速运动场景
- 编码格式：H.264 / H.265
- 最大文件大小：4GB
- 最大时长：15 分钟（但建议控制在 60 秒以内）

## 文字叠加最佳实践

- 大号加粗文字（相当于 48-72px）
- 居中或置于画面上三分之一
- 高对比度（白色文字加黑色阴影）
- 仅放关键短语——不要放完整句子
- 文字动画出现效果能提升互动率

## 抖音节奏模板

**教程型（Tutorial，30-60s）：**
```
[Hook: 成品展示 3s] → [问题引出 3s] → [步骤1 5-8s] → [步骤2 5-8s] → [步骤3 5-8s] → [成品特写 3s] → [CTA 3s]
```

**故事型（Story，30-60s）：**
```
[Hook: 冲突/悬念 3s] → [背景交代 5s] → [发展 10-15s] → [高潮/反转 5-8s] → [结局 3-5s] → [CTA 3s]
```

**Vlog型（Vlog/日常，45-90s）：**
```
[Hook: 今日亮点 3s] → [场景1 10-15s] → [过渡 2s] → [场景2 10-15s] → [场景3 10-15s] → [总结感悟 5s] → [CTA 3s]
```

## 抖音输出格式：短视频发布信息

```markdown
### 抖音
- **标题:** [平台优化标题——简短、有力、引发好奇]
- **标签:** #tag1 #tag2 #tag3 #tag4 #tag5
- **文案:** [发布文案]
- **发布时间建议:** [建议发布时间]
```

## 抖音输出格式：图文发布信息

```markdown
### 抖音（如适用）
- **标题:** [简短有力的标题]
- **文案:** [简短文案]
- **标签:** #tag1 #tag2 #tag3 #tag4 #tag5
```

## 抖音平台注意事项

- 封面图/首帧尺寸：9:16（1080×1920）
- 抖音偏好完播率高的内容——完播率是关键的算法信号
- 为文字叠加预留空间（特别是画面的上三分之一或下三分之一）
- 热门 BGM 能显著提升算法推荐——始终检查是否有相关的热门音频可用
