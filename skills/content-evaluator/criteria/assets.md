# 素材阶段评审标准

## 评审维度

### 1. 技术质量 (technical_quality)
检查方法：使用 ffprobe 和 Read 工具

**图片：**
- 分辨率是否符合要求（竖屏≥1080x1920，方形≥1080x1080）？
- 是否有明显的模糊、噪点、色块等AI伪影？
- 文件格式是否正确（PNG/JPG）？

**视频：**
- 分辨率是否符合要求？
- 帧率是否≥24fps？
- 是否有音频轨（重要！）？
- 编码是否为 H.264/H.265？
- 时长是否符合分镜规划？

```bash
# 必须运行的检查命令
ffprobe -v error -show_entries stream=width,height,codec_name,r_frame_rate,codec_type -of json {file}
```

**评分标准：**
- 9-10: 所有技术指标完美达标
- 7-8: 基本达标，个别参数可优化
- 5-6: 有技术问题但不影响发布（如分辨率略低）
- 1-4: 严重技术问题（无音频、分辨率严重不足、文件损坏）

### 2. 美学质量 (aesthetic_quality)
检查方法：使用 Read 工具查看图片，评估视觉效果

- 构图是否合理（主体突出、空间平衡）？
- 色彩是否和谐（不过饱和、不发灰）？
- 光线是否自然？
- 整体视觉感受是否专业？
- 是否存在典型的"AI味"（过度平滑、塑料感、不自然的光泽）？

**评分标准：**
- 9-10: 视觉效果卓越，堪比专业摄影/设计
- 7-8: 视觉效果良好，适合社交媒体发布
- 5-6: 可以接受但缺乏美感
- 1-4: 美学质量差，不适合发布

### 3. Prompt遵从度 (prompt_adherence)
- 生成结果是否匹配分镜描述？
- 角色外观是否一致？
- 环境/场景是否正确？
- 动作/姿态是否正确？

**评分标准：**
- 9-10: 完美匹配所有描述要素
- 7-8: 主要元素匹配，细节略有偏差
- 5-6: 基本方向对，但部分要素不匹配
- 1-4: 与描述严重不符

### 4. 风格一致性 (style_consistency)
- 所有素材的色调是否统一？
- 角色在不同镜头中是否看起来一致？
- 风格关键词是否贯穿始终？
- 光线感觉是否前后一致？

**评分标准：**
- 9-10: 高度一致，像是同一组拍摄的
- 7-8: 基本一致，个别素材略有差异
- 5-6: 有明显的风格跳跃
- 1-4: 每张素材看起来像不同风格

### 5. 素材完整性 (completeness)
- 分镜脚本中的所有镜头是否都有对应素材？
- 首帧和视频片段是否都已生成？
- 是否有遗漏的素材？

**评分标准：**
- 9-10: 所有素材齐全，还有备选方案
- 7-8: 所有必需素材齐全
- 5-6: 缺少个别非核心素材
- 1-4: 大量素材缺失

## 视觉审查清单

对每个图片/视频文件，依次检查：

1. □ 主体是否完整、清晰？
2. □ 手部/手指是否正常（常见AI问题）？
3. □ 面部是否自然（非畸变）？
4. □ 文字是否正常（如有）？
5. □ 背景是否干净、无杂乱元素？
6. □ 色彩是否和谐？
7. □ 是否有拼贴/重复/异常的图案？

## Critical 问题示例
- 视频文件无音频轨
- 图片分辨率低于720p
- 角色面部严重畸变
- 素材数量严重不足（缺少超过50%的镜头）
- 所有素材风格完全不一致（每张图看起来是不同世界的）
