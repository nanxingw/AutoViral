# 抖音 (Douyin) 平台参考

## 算法机制

抖音采用**阶梯式流量池系统**：

1. **初始流量池（200-500 播放）：** 新内容进入小规模测试池。算法衡量以下指标：完播率、点赞率、评论率、转发率、关注率。
2. **二级流量池（3,000-5,000 播放）：** 如果指标超过阈值（完播率 >30%、点赞率 >3%、评论率 >1%），内容晋级。
3. **三级流量池（10,000-50,000 播放）：** 需要更高的指标门槛。在这一层停滞的内容通常是开头钩子不错但中间或结尾较弱。
4. **爆款流量池（10万+ 播放）：** 算法推送至更广泛的用户群体。在这个阶段，转发率成为最关键的信号。

**核心排名信号（按权重排序）：**
- 完播率 — 最重要；完整看完的视频排名最高
- 互动率 — 评论的权重高于点赞
- 转发率 — 代表高价值内容
- 关注率（关注转化）— 体现创作者权威性
- 复播率 — 代表内容吸引力强

**发布时间：** 在 12:00-13:00 或 18:00-22:00 发布，初始流量池表现最佳。周末生活类内容互动率高出 20-30%。

## 爆款机制

**3 秒定律：** 如果观众在前 3 秒没有被吸引，就会划走。有效的开头钩子：
- 悬念式："你绝对想不到..." / "看到最后我惊了..."
- 冲突式：出人意料的对比、前后反差
- 利益式（价值承诺）："学会这个，月入过万" / "3个技巧让你..."
- 共鸣式：情感触发 — 怀旧、不平、自豪
- 视觉冲击：第一帧就给出震撼画面

**短视频节奏结构：**
- 0-3秒：钩子（悬念/冲击）
- 3-15秒：制造紧张感，输出第一个价值点
- 15-45秒：核心内容，每 5-7 秒变换一次节奏
- 最后 3-5秒：行动号召（CTA）— "关注我" / "评论区留言" / 反转结尾

**驱动分享的情感触发点：**
- 实用价值 — 引发"收藏了"的反应
- 情感共鸣 — 感情、家庭、职场相关
- 社交货币 — "这个太有趣了必须分享"
- 身份认同 — "这不就是我吗"

## 标签策略

使用**金字塔结构**（3 层）：
1. **大标签（10亿+ 播放）：** 1-2 个标签用于增加曝光（如 #美食 #生活）
2. **中标签（1000万-10亿 播放）：** 2-3 个标签用于品类定位（如 #家常菜 #一人食）
3. **小标签（<1000万 播放）：** 2-3 个标签用于精准排名（如 #上班族快手菜 #10分钟早餐）

**规则：**
- 总计不超过 5-7 个标签
- 最重要的标签放在最前面
- 如有相关的热门挑战赛标签，加入 1 个
- 避免使用被限流或违规的标签

## 数据获取工具

本 skill 自带 Python 脚本工具，位于 `skills/trend-research/scripts/` 目录：

### `douyin_hot_search.py` — 抖音热搜直接获取
直接调用抖音 Web API，获取实时热搜榜（含热度值、标签）。
```bash
# JSON 输出
python3 skills/trend-research/scripts/douyin_hot_search.py

# 表格输出，前20条
python3 skills/trend-research/scripts/douyin_hot_search.py --format table --top 20
```

### `newsnow_trends.py` — 多平台热搜聚合
通过 NewsNow API 一次获取多个平台的热榜数据（抖音、微博、知乎、百度、B站等 50+ 平台）。
```bash
# 获取抖音+微博+知乎
python3 skills/trend-research/scripts/newsnow_trends.py douyin weibo zhihu

# 获取所有主要中文平台
python3 skills/trend-research/scripts/newsnow_trends.py --all-cn --format table --top 10

# 查看支持的平台列表
python3 skills/trend-research/scripts/newsnow_trends.py --list
```

### 广度模式脚本用法
```bash
# 抖音热搜（直接API，有热度值）
python3 skills/trend-research/scripts/douyin_hot_search.py --top 30

# 多平台交叉对比（发现跨平台热点）
python3 skills/trend-research/scripts/newsnow_trends.py douyin weibo zhihu baidu --top 15
```

### 深度模式脚本用法
```bash
# 抖音热搜（检查目标话题是否在榜、当前热度）
python3 skills/trend-research/scripts/douyin_hot_search.py --top 30
```
