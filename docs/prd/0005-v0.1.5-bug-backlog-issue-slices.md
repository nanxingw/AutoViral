**类型**：HITL（需产品决策）· **severity**：high · **triage**：ready-for-human · **class**：stale-served-as-live（[[project_explore_live_dot_contradiction]] #82 的兄弟）

**Code-area hints**：`src/server/routes/trends.ts`、`src/server/.../normalize.ts:122`、`web/src/queries/trends.ts`(SUPPORTED_REFRESH_PLATFORMS)、`web/src/features/explore/TrendingPanel.tsx`。

### B4 · WorkCover 的 `failed` 错误态在封面 URL 变化时不重置 —— 一次瞬时 404 后永久卡在灰渐变
**类型**：AFK · **severity**：medium · **triage**：ready-for-agent · **class**：no-auto-refresh / built-not-wired（B3 的近亲）

**What**：`web/src/features/works/WorksGrid.tsx:157-210` WorkCover 内 `const [failed] = useState(false)`，img/video `onError` 置 true，但**无** `useEffect(... ,[cover])` 重置、media 无 `key={cover}`。守卫 `if (!cover || failed) return <fallback>` 在渲染 img/video 前短路 → useWorks refetch 来的新封面永不挂载、onError 无从再触发清 `failed`；父层仅 `key={w.id}` → failed 跨 refetch 持久。触发：agent 后台渲染 → 首个封面 URL 写盘瞬间 404 → setFailed(true) → agent 完成、refetch 附真封面 → 卡片永远灰渐变直到整页 reload。

**Acceptance criteria**：
- [ ] WorkCover 内加 `useEffect(() => setFailed(false), [cover])`，**或** `<img>/<video>` 上加 `key={cover}` 使其随 URL 变化 remount 重试。
- [ ] 单测：mount 时 cover 404→failed，rerender 换有效 cover → 重新尝试渲染该 cover（非停在 fallback）。

**Code-area hints**：`web/src/features/works/WorksGrid.tsx`(WorkCover)。
**Blocked by**：None（与 B3 同页，可同片处理）。

### B5 · `duplicateSlide` 用 `Date.now()` 单独生成 slide+layer id，绕过共享的防碰撞 `uid()` —— 同毫秒复制产出字节相同 id
**类型**：AFK · **severity**：medium · **triage**：ready-for-agent · **class**：react-key / id-collision（本会话 trending key 碰撞的兄弟 + dup-id-gen）

**What**：`web/src/features/editor/store.ts:82-97`（具体 88-93）`copy.id` 与各 layer id 的唯一性仅来自 `Date.now().toString(36)` + per-layer index `i`，**无单调计数器** → 同一毫秒内两次 `duplicateSlide(sameId)` 产出相同 copy.id 与相同 layer ids。后果：React key 碰撞（Stage.tsx:74 / SlidesNav.tsx:185）；`updateLayer`(find-first-then-break)改第 2 个副本时命中第 1 个；`removeLayer`(filter ALL slides)删两份。防碰撞生成器 `genLayerId`(计数器式)已由 `@shared/carousel` 导出、`editor/types.ts:11` 再导出却未用。**潜伏/间歇**：真人双击通常 >1ms（Date.now 不同，不触发）；仅快速双击被记为两次点击 / 任何程序化调用 同毫秒才触发，触发时静默污染持久化 carousel。

**Acceptance criteria**：
- [ ] `duplicateSlide` 改用共享计数器生成器：`copy.id = genSlideId()`、layers `genLayerId()`（经 `@shared/carousel` 既有再导出）。
- [ ] fake-timers 测试：冻结 `Date.now()` 下两次 `duplicateSlide` 同一 slide，断言 4 个 id（2 slide + 各 layer）全互异。

**Code-area hints**：`web/src/features/editor/store.ts`、`src/shared/carousel.ts:139-146`(uid/genLayerId)、`web/src/features/editor/types.ts`。
**Blocked by**：None。
